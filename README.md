# Wayback Availability Checker

Simple static web page that queries the Internet Archive Wayback Machine Availability API and displays the closest archived snapshot for a given URL.

How to use
- Open `index.html` in a browser (no server required).
- Enter a URL (for example `https://example.com`).
- Optionally enter a timestamp in `YYYYMMDDhhmmss` format to request a snapshot near that time.

Deploy
- Host the files with GitHub Pages, Netlify, or any static host. For GitHub Pages, push this directory to a repository and enable Pages for the `main` branch.
